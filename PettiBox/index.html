PettiBox: Bookmark Manager

PettiBox is an Android app developed by GhostGram Labs.

Save and organize links, notes, images, screenshots, PDFs, and files in one place.

Features

  • Save from browser and apps using Android share
  • Collections and tags
  • Favorites and archive
  • Reminders
  • Image and PDF text recognition
  • Fast search
  • Optional Google Drive backup and restore

Google Drive Usage

Google Drive access is optional and used only for PettiBox backup and restore. PettiBox does not access unrelated Google Drive files.
